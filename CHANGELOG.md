# Changelog

## [1.0.0] - 2026-06-15

### Added
- Initial release of Pixel Studio
- Drawing tools: Pen, Eraser, Fill Bucket
- Color picker with custom color selection
- Predefined color palette
- Grid canvas with adjustable size
- Zoom functionality
- Dark theme UI
- Responsive design for multiple screen sizes

### Features
- PixelStudio/index.html - Main interface
- PixelStudio/style.css - Dark theme styling
- PixelStudio/js/App.js - Application controller
- PixelStudio/js/Grid.js - Canvas management
- PixelStudio/js/Pixel.js - Pixel data structure
- PixelStudio/js/Tool.js - Tool implementations

### Technical Stack
- HTML5 Canvas API
- Vanilla JavaScript (ES6)
- CSS3 Grid and Flexbox
